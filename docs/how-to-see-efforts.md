# 如何看待努力和成功的关系？
## 一、试问一下自己
1. 在面对考试，面对一个个充满竞争的挑战的时候，你们的内心是怎么想的，转而影响到你个体的情绪情感变化是怎么样的？
2. 有没有因为面对考试之类的事情，而让你产生了很多想法，转而这些想法影响到了你的生活
3. 或者说在这样想法的基础上，你就算是赢了，那你真的开心吗？你到底得到了什么呢？
4. 那如果失败了呢，你会怎么样呢？你可以接受你的失败吗？
### 我认为，你们已经有答案了，虽然我并不清楚你们是怎么想的。
## 二、我的想法
1. 一心想赢，充满焦虑情绪，长此以往，不仅影响到了个人的真正的生活，甚至会影响健康。
2. 当临场发挥，会因为情绪导致对自己行为的后悔，因为你会认为自己没有发挥出正常的能力，因为你受到了自我的干扰。
3. 就算是真的赢了，你会感受到短暂的兴奋，但是你并没有让自己真正享受获得知识和能力的过程，并没有领会到考试真正带来的个人幸福感的提升。
4. 转而如果失败了，那么你会错误理解努力导致成功的归因观，甚至产生习得性无助
## 三、总结
**用一位运动员的话来总结这种心态的差别：**
- 如果我的目标是“赢得金牌”，那么我的**命运在裁判亮出分数的那一刻就结束了**。它可能是狂喜，也可能是深渊。
- 但如果我的目标是**“完成一套我能力范围内最极致的动作”**，那么无论裁判给出什么分数，在我稳稳落地的刹那，我就已经**赢得了我对自己最完整的交付**。那份内心的“通透”与“完成感”，是任何金牌都**无法替代的**，也是任何人**无法夺走的**。

**所以，面对一次又一次的考试，请抱持“通过”的信念：
穿越它，去往你想去的未来；
借助它，看清并塑造你想成为的自己。
这，才是永不枯竭的动力，和真正坚实的立足之地。**
